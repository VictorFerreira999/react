# Configurações do TypeScript

1. Inicializart um projeto node.js usando o npm init

2. Instalar o TypeScript
   2.1 instação global: `npm install -g typescrypt`
   2.2 instalação local (dentro apenas do projeto): `npm install typescript -D`
3. Utilizar o TypeScript instalado para traspilar para JS usando `npx tsc index.ts --watch` - para todoas as mudanças ocorram automaticamente
